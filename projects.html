Sony Pictures Imageworks - The Reboot Project

Objective: To monitor and track reboot activity across all user machines to identify issues and optimize system performance.

Technologies Used: Python for scripting, Linux commands for system monitoring, and Splunk for data visualization.

Approach:

  • Developed a script that utilized Linux commands to log reboot activities.
  • Integrated the script with the Internet Control Message Protocol (ICMP) to ensure real-time monitoring and alerting.
  • Used Splunk to aggregate and visualize the data, making it easier to spot trends and problematic machines.

Outcome: Enhanced technical support processes by quickly identifying machines that required attention, leading to more proactive maintenance and reduced downtime.
